# Завдання №1
### Опис завдання
Компанія виробляє два види напоїв: "Лимонад" і "Фруктовий сік". Для виробництва цих напоїв використовуються різні інгредієнти та обмежена кількість обладнання. Задача полягає у максимізації виробництва, враховуючи обмежені ресурси.

#### Умови завдання:
1. "Лимонад" виготовляється з "Води", "Цукру" та "Лимонного соку".
2. "Фруктовий сік" виготовляється з "Фруктового пюре" та "Води".  
3. Обмеження ресурсів: 100 од. "Води", 50 од. "Цукру", 30 од. "Лимонного соку" та 40 од. "Фруктового пюре".
4. Виробництво одиниці "Лимонаду" вимагає 2 од. "Води", 1 од. "Цукру" та 1 од. "Лимонного соку".
5. Виробництво одиниці "Фруктового соку" вимагає 2 од. "Фруктового пюре" та 1 од. "Води".

### Виконання
Фактично маємо систему лінійних рівнянь:
- `2x + y ≤ 100`;
- `x ≤ 50`;
- `x ≤ 30`;
- `2y ≤ 40`;
- `x, y ≥ 0`.
 
 Мета: максимізувати `x`+`y` 

Для вирішення задачі використовуюється бібліотека `PuLP`, покроково:
1. Імпорт бібліотеки `PuLP`
2. Створення задачі максимізації.
3. Оголошення змінних.
4. Накладання обмежень ресурсів.
5. Визначення цільовою функції.
6. Розв`язання залачі.

### Результат
Максимальна кількість продукції — 50 одиниць (30 лимонаду й 20 соку).  
Цей код демонструє можливості лінійного програмування для оптимізації виробництва з обмеженими ресурсами.

# Завдання №2
### Опис завдання 
Завдання полягає в обчисленні значення інтеграла функції методом `Монте-Карло`. Також потрібно перевірити правильність розрахунків, щоб підтвердити точність методу Монте-Карло, шляхом порівняння отриманого результату та `аналітичних розрахунків` або результату виконання функції `quad`.

### Умови завдання
1. Функція, яку потрібно інтегрувати:
`f(x) = x^2`
2. Діапазон інтегрування:
   - a = 0  -  нижня межа;
   - b = 2  -  верхня межа.

### Виконання
1. Функція та межі інтегрування:  
Визначено f(x)=x^2 та межі інтегрування:
   - a=0, 
   - b=2.
2. За допомогою matplotlib побудовано графік f(x) на розширеному діапазоні.
3. Метод Монте-Карло:
   - Генерується велика кількість випадкових точок (в цьому прикладі - 10000 точок). 
   - Визначається, які точки лежать під кривою.
   - Оцінка площі: `площа = (b−a) * f(b)`⋅частка точок під кривою.
4. Аналітичний розрахунок.
5. Чисельний розрахунок (quad): використовується функція quad з бібліотеки `SciPy` для точного чисельного інтегрування.
6. Відображення точок на графіку.   
Точки Монте-Карло відображаються: 
   - сині — під кривою, 
   - червоні — над кривою.   
Це дозволяє візуально оцінити точність методу.
7. Вивід результатів  
Порівнюються всі три значення площі, а також обчислюється похибка методу Монте-Карло відносно аналітичного результату.

### Результат
| Метод                | Значення       | Похибка           |
|----------------------|----------------|-------------------|
| Метод Монте-Карло    | 2.6632         | 0.0035            |
| Аналітичний метод    | 2.6667         | 0                 |
| Quad-метод           | 2.6667         | 2.96e-14          |


### Висновки
- Метод Монте-Карло добре підходить для оцінки площі складних фігур, але його точність залежить від кількості точок.

- Візуалізація дозволяє зрозуміти, як працює метод Монте-Карло: чим більше точок, тим точніша оцінка.

- Порівняння з аналітичним та чисельним результатами показує, що всі методи дають близькі значення.
